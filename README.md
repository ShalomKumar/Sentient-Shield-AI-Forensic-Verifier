# 🛡️ Sentient Shield: AI Forensic Verifier & Cyber Terminal

An advanced, full-stack cyber forensics and threat intelligence terminal designed to detect malicious URLs, calculate real-time cyber risk scores, analyze phishing vectors, and ensure strict compliance with the **Information Technology Act (IT Act) 2000**.

---

## 🚀 Key Features

* **Real-time Link Forensic Analysis:** Scans and breaks down suspicious URLs to intercept phishing vectors.
* **Cyber Risk Scoring Engine:** Calculates a dynamic threat level score based on live domain signals.
* **IT Act 2000 Compliance Guard:** Automatically flags legal violations and identifies specific sections under Indian Cyber Law.
* **Secure Forensic Reports:** Generates automated cyber logging and downloadable threat verification reports.
* **Hardened Admin Dashboard:** Multi-layered admin terminal to monitor suspicious network logs and attack histories.

---

## 🛠️ Tech Stack Used

* **Backend Engine:** PHP (Full Object-Oriented Architecture)
* **Database & Integrity:** MySQL / SQL (Structured Threat Logging)
* **Frontend Terminal:** HTML5, CSS3 (Custom Cyberpunk Theme), JavaScript (Async API calls)
* **Environment:** XAMPP Localhost Terminal

---

## 📁 Repository Structure

* `admin/` - Forensic monitoring panel and report overview dashboards.
* `core/` - The brain of the project containing the `analyzer.php`, `blacklist.php`, and `cyber_law_helper.php`.
* `config/` - Secure core database connectivity engines (`db_connect.php`).
* `reports/` - Automated database logs and generated assessment templates.
* `sql/` - Clean database schemas containing cyber intelligence configurations.
